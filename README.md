Nova背单词
====

这是一个简单的背单词系统。主要功能包括：

 - 多用户
 - 词库和单元管理
 - 单元学习
 - 单元复习
 - 测试

要展示的页面包括：

 1. 用户登录
 2. 新用户注册
 3. 重置密码
 4. 修改密码
 5. 用户激活
 6. 修改用户信息
 7. 学习情况概况
 8. 查看用户
 9. 新建群组
 10. 查看群组
 11. 编辑群组
 12. 查看单词
 13. 搜索单词
 14. 新建词库
 15. 列出词库
 16. 查看词库
 17. 编辑词库
 18. 查看单元
 19. 编辑单元
 20. 单元学习
 21. 单元复习
 22. 学习记录展示
 23. 易错单词展示
 24. 查看所有测试
 25. 新建测试
 26. 编辑测试
 27. 进入测试
 28. 查看测试结果
 29. 查看测试统计情况

以后可能还会增加新的页面。

## 开发环境 ##

程序使用的开发工具是 [pycharm](https://www.jetbrains.com/pycharm/)。
